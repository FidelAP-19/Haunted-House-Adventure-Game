# 👻 Haunted House Adventure

A haunted-house "Choose Your Own Adventure" game with both:
- a Node.js CLI version
- a browser GUI version (vanilla HTML/CSS/JS)

## Run (CLI)

```bash
node index.js
```

## Run (Browser GUI)

Serve the folder with a local static server, then open `index.html`:

```bash
python3 -m http.server 8000
```

Then visit: `http://localhost:8000/index.html`

## Built with
- Node.js
- Vanilla HTML/CSS/JS
- GitHub Copilot (AI-assisted code generation)

## Scenes
- Entrance → Hallway → Library / Basement / Attic → Escape or Death
